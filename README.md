# Simply GIPHY Gallery with React and JSS

Open [https://react-giphy-gallery.netlify.app/](https://react-giphy-gallery.netlify.app/) to view the project.

## Scripts

### start
```bash
yarn start
```

### test
```bash
yarn test
```

### build
```bash
yarn build
```

### eject
```bash
yarn eject
```
